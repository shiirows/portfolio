Qu'est-ce que la Wild code school ?

La Wild Code School propose des formations intensives aux métiers tech - Développement web, Data analyse, Cybersécurité, Product management - sur campus, à distance ou en entreprise.

Durant cette formation, j'ai beaucoup appris sur le métier de développeurs web. J'ai appris à utiliser les outils de développements, a utiliser diffèrent langages et Framework.
Langages : HTML, CSS, Java, JavaScript, Typescript.
Framework : Angular, Node.js, Spring.
